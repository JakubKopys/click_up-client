## ClickUp::Client

Unofficial ClickUp API Client for Ruby. Development in progress.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'click_up-client'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install click_up-client
